introduction

Введение

JavaScript — это кросс-платформенный объектно-ориентированный язык сценариев.

Он мал и легок, и в среде хоста (например, веб-браузере) JavaScript может управлять объектами своего окружения.

hello world

Hello world

Для начала работы с JavaScript, напишите следующий код:

function greetMe(yourName) { alert("Hello " + yourName); }

Вызовите функцию:

greetMe("Мир");
variables

Переменные

Переменные используются как символические имена для значений в вашем приложении.

Вы можете объявить переменные несколькими способами:

  • С помощью ключевого слова var.
  • Простым присваиванием значения (что не рекомендуется).
  • С помощью ключевого слова let.

Каждый из этих методов имеет свои особенности и область применения.

Обязательно выбирайте подходящий способ в зависимости от контекста.

data types

Типы данных

JavaScript поддерживает несколько типов данных, включая:

  • Boolean: true и false.
  • null: специальное значение, обозначающее отсутствие значения.
  • undefined: значение не определено.
  • Number: целые и дробные числа.
  • String: последовательности символов.

Каждый из этих типов данных имеет свои особенности и способы использования.

functions

Функции

Функция определяется с использованием ключевого слова function.

Пример простой функции:

function square(number) { return number * number; }

Функции могут принимать параметры и возвращать значения.

Это позволяет создавать более модульный и читаемый код.
